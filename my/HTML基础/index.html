HTML基础

发表于 2019-03-21 | 分类于 HTML

因为随着后端写项目，那么不可避免的需要一个人写网页相关的东西，在加上一开始自学的时候没用心记录导致了这方面知识的薄弱所以写下了这一篇文章以供自己温习，且让人快速找到重点。
所以这是一篇精简的HTML一些标签基础。

HTML

首先来一个最最最基本的模板

<!DOCTYPE html>  告诉游览器这是html5版本
<html lang="en"> 
<head>
<meta charset="UTF-8"> 
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">
<title>Document</title>
</head>
<body>

</body>
</html>

html

标签限定了文档的开始点和结束点

head

用于定义文档的头部

mate

提供有关页面的元信息(如针对搜索引擎和更新频度的描述和关键词)

title

定义文档的标题

link(单标签)

专门用于加载样式表

script

专门用于加载Js

body

h1,h2…h6

用于定义标题

small

用于定义副标题

p

段落标签

div

无属性标签 就像一个空盒子一样

a

锚标签 用于跳转指定网页
href指定地址

img(单标签)

图片标签展示 图片的标签
src 指定图片地址 可以网络可以本地
alt 当图片无法加载时候显示 alt内容

hr(单标签)

水平分割线

table

用于创建一张表

tr

table row 定义表行

th

table head 定义表头

td

table body 定义表身体

thead

加上此标签 增加 阅读性 包裹 th用

tbody

加上此标签 增加 阅读性 包裹 tb用

header

代表语义 标签定义文档的页眉（介绍信息）

footer

代表语义 标签定义文档或节的页脚 (通常包含文档的作者、版权信息、使用条款链接、联系信息等等)

form

用于为用户输入创建 HTML 表单。常见属性action
当要上传文件时候

button

按钮。单独存在意义不大，一般配合提交用

input

输入框 很多种样式

abbr

专门为缩写的标签

code

行内标签写代码展示用

pre

对比code 适用于一大段代码

注意事项

块标签
  1.每一个快属性标签都是从新的一行开始，而且之后的元素也都会从新的一行开始
    （因为每一个块属性标签都会直接占据一整行的内容，导致下面的内容也只能从新的一行开始）
  2.块属性标签都是可以设置宽度、高度，行高，距顶部距离，距底部距离
  3.块属性标签的宽度假如不做设置，会直接默认为父元素宽度的100%
  4.块属性标签是可以直接嵌套的
  5.p标签中不能嵌套div标签
  块标签有
  <div>、<p>、<h1>…<h6>、<ol>、<ul>、<li>、<address>、    <blockquote>、<form>
行内标签
1.行属性标签它和其它标签处在同一行内
2.行属性标签无法设置宽度，高度 行高 距顶部距离 距底部距离
3.行属性标签的宽度是直接由内部的文字或者图片等内容撑开的
4.行属性标签内部不能嵌套行属性标签（a链接内不能嵌套其他链接）
行内标签有
<a>、<span>、<i>、<em>、<strong>、<label>、<q>、<var>、<cite>、<code>
