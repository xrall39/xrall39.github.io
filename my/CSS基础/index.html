选择器

没有选择器就没法确定样式的作用范围，毫无重点的加样式，最后只会是一锅粥。所以熟练使用选择器是必备技能。

核心选择器

  • .(英文句号)

    • 类选择器，使用类名选择。可能是使用最多，也是最灵活的选择器了。
    • 如：
      • HTML: <div class="red">类</div>
      • CSS: .red { ... }
        • .red 就是类选择器，由于类可以在页面中重复出现，它天生就拥有批量选择的能力。

  • #(井号)

    • ID选择器，使用ID选择
    • 如：
      • HTML：<div id="logo">ID</div>
      • CSS：#logo { ... }
        • #logo就是ID选择器，由于ID不可以在页面中重复出现，它天生就拥有极高权重，也就是说如果类选择下定义的样式和ID选择器下定义的样式发生冲突后一定是ID选择器获胜。

  • div(元素名称)

    • 元素选择器，直接使用元素名称选择。
    • 如
      • HTML：<div>元素</div>
      • CSS：div { ... }
        • 一般与.或#结合使用。

选择器权重（优先级）

选择器权重(就是优先级)
一个元素在页面中越特殊权重越高。
!important(基本不用) > 内联 > ID选择器 > 类选择器/属性选择器/伪类 >元素选择器
谨慎使用内联 style

核心选择方式

  • 目标1 目标2 ... （中间有空格）
    • 后代选择器，可以将范围缩小至下一个元素。
    • 目标之间可以是任何层级的包含关系，如父子关系，爷孙关系，不适用于兄弟关系(兄弟就是同级)。
  • 目标1 > 目标
    • 父子选择器，可以将范围缩小至儿子辈，但选不中孙子辈。
    • 如：
      • <div>这 <strong> 是<p> 一个</p></strong> 测试</div>
      • div > p {...} 上面这种是无法选择的因为中间还有个strong

相邻（兄弟）选择器

.a + div   a类相邻的 div 标签
.a ~ div  a类相邻以下的标签
以上有一个问题是 他不会选择 .a 上面的div 程序只会往下识别

伪类选择器

:隔开   button : hover 意思鼠标放上去的样式
button:active 点击时特效
:focus 聚焦 焦点激活
:visited 点击后的样式

:after 操作文档内容最后
:before 操作文档内容开头 content="*" 开头就会添加*
div p:neh-child(odd) 给div下第几个p元素修改样式

[] 包裹为属性选择器
[title] {
}
还可以[title="XX"] 指定title 属性等于XX 的才有效果
*="XX" 模糊匹配XX      $="XX"以XX结束的 
^="XX" 以XX开头

核心属性

  • display(显示方式)

    • block
      • 块，默认占整宽。
      • 通常作为容器分割页面，一些元素默认显示为block，如<div>或<p>。
    • inline
      • 行内，通常存在于块级元素中作为内容填充，如文章的文字，图标，链接。一些元素默认显示为inline，如<span>或<a>。
    • inline-block
      • 行内块。
        前一篇博客有讲inline和inline-block区别 行属性标签无法设置宽度，高度 行高 距顶部距离 距底部距离 行内块则无限制。

  • float (浮动)

    • 注意：
      • 浮动元素撞到父级元素的“内壁”或和其他浮动元素摞到一起时才会停止“移动”，如果父级元素的空间不够，它们就另起一行。
      • fixed或absolute元素不浮动。

字体属性

font-family: 字体
font-weight: 字体宽度 （加粗）
font-size：字体大小

text-align：文字对齐方式
line-height：文字行高
text-decoration：文字装饰（下划线等）

dispaly属性

dispaly:  用于指定元素显示的方式
    none 用于隐藏元素
    block 块级元素
    inline 行内元素 上下操作无效
    inline-block 行内块元素 常用于 按钮 导航栏等

margin（外边距）
border（边框）
padding（内边距）
content（内容区）就是宽高 height width
